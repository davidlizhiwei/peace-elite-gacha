# 🎁 和平精英 - 皮肤盲盒

和平精英模拟抽奖游戏，体验皮肤盲盒抽奖乐趣！

## ✨ 功能特性

- **单抽/十连抽** - 支持单次抽取和十连抽取
- **充值系统** - 模拟充值钻石和抽奖券
- **真实图片** - 使用 picsum.photos 展示皮肤图片
- **收藏系统** - 自动收集获得的皮肤
- **稀有度系统** - 普通、稀有、史诗、传说、神话五种稀有度

## 🎮 游戏玩法

1. 点击"充值"按钮获取钻石或抽奖券（模拟充值）
2. 点击"单次抽取"或"十连抽"进行抽奖
3. 单抽消耗：💎 100 或 🎫 1
4. 十连抽消耗：💎 900 或 🎫 8（享受折扣）
5. 收集所有稀有皮肤！
